Packagecom.segfaultlabs.swfutils.ABC
Classpublic final class ABCcodedump

Class used to illustrate how code form doABC tag should be dumped. Code based on dumpabc.as which is a part of a Tamarin project. This should (will) be developed to dump human readable code, not only 'assembly' notation.
This class defines all AVM2 opcodes used in byte code (possibly some fp10 opcodes are missing).
