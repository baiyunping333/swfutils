public function parse(data:SWFDataInput, abc:ABCinfo, cpool:ABCCPool, _logWriter:ILogWriter = null):void

Method for parsing doABC code block. All parse out data is stored in ABCInfo instance, and all constants used in block are stored in a ABCCPool object. Constant pool is common for all doABC blocks in file, this is why it is stored in seperate object. Function can also write out some debug information (if compiled with -define=sflabs::debug,true compilation flag) if log writer object is specified.

Parameters
data:SWFDataInput — - swf file reading stream
 
abc:ABCinfo — - ABCinfo what will store all data from parsed doABC block
 
cpool:ABCCPool — - ABCCPool, constant pool used in parsed file
