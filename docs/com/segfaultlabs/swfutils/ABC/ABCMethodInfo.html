Single method, function object description. This class stores basic information about functions (methods). Code is stored in related ABCMethodBody object, and can be accessed throught body property. ABCMethodInfo are defined for both annonimous and well defined functions. The difference is, that for annonimous functions body property is empty. That is code is unknown at this level, it is defined at runtime.
