InheritanceABCInstance ABCTraits

This class describes a single Action Script 3 class, or more precisely a single class instance. ABCInstance stores information about all non static properties and methods of the class. Static properties and methods are stored in ABCClass obejct, that can be accessed using the cref property of ABCInstance. ABCInstance together with related ABCClass are fully describing single Action Script class, including information about metadata related with clas and/or with properties.
