Packagecom.segfaultlabs.swfutils
Classpublic final class SWFDataInput
Implementsflash.utils.IDataInput

This class is used to read data from SWF file. Currently this class can read all (or almost all) SWF specific types. For a complete list of SWF types refere to SWF file format documentation.
