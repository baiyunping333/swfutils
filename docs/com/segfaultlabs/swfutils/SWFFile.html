hasActions()method 
public function hasActions(tagArr:Array, checkSprites:Boolean = true):Boolean

Check if there are any actions in swf file. Valid only for AVM1. In case of AVM2 code is always present in file. If second parameters is set true, method also scans alll sprites present in file.

Parameters
tagArr:Array — tags array readed with SWFFile object
 
checkSprites:Boolean (default = true) — if true function checks all defined sprites

Returns
Boolean — if return value is true then reader contains swf file with actions
